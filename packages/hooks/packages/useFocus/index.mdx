## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useFocus } from '@snappmarket/hooks';

const SampleComponent = props => {
  const focusRef = useFocus(null);

  return (<input type="text" ref={focusRef}/>)
};
```
