## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useTimeout } from '@snappmarket/hooks';

const SampleComponent = props => {
  useTimeout(() => {
    // do a callback
  }, [200]);
};
```
