## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useTimer } from '@snappmarket/hooks';

const SampleComponent = props => {
  const { time, start: startTimer, reset: resetTimer } = useTimer({
    endTime: 0,
    initialTime: 100,
    timerType: 'DECREMENTAL',
  });
};
```
